## Import

1. Add it in your root build.gradle at the end of repositories:

        allprojects {
            repositories {
                ...
                maven { url 'https://jitpack.io' }
            }
        }

2. Add the dependency in your module build.gradle:

	dependencies {
	        implementation 'com.github.dyguests:XXXBanner:x.y.z'
	}
